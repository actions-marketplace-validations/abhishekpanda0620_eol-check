## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please open a new issue in this repository.

### Process

1. **Report**: Open a new issue with a clear description of the vulnerability.
2. **Label**: Please add the `security` label if possible.
3. **Fix**: We will work on a fix and release a patch.

Thank you for helping keep `eol-check` secure!
